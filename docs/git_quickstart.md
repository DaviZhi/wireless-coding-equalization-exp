# Git 快速入门

## 常用流程

```bash
git status
git add .
git commit -m "Complete channel coding and equalization experiment"
git push origin main
```

## 创建 PR

1. 打开自己的 GitHub 仓库。
2. 点击 Pull requests。
3. 选择教师仓库作为 base。
4. 创建 PR 后等待自动评分。

## 重新触发评分

- 修改代码后重新 push。
- 或关闭 PR 后重新打开。
- 不建议直接 re-run 旧 workflow。
