# 信道均衡实验原理

## 1. 为什么需要均衡

宽带无线信道存在多径传播，不同路径的延迟会导致符号间干扰（ISI）。当符号周期小于或接近多径时延扩展时，ISI 会成为主要误码来源。

均衡器的目标是补偿信道失真，使等效信道尽量接近冲激响应。

## 2. 多径信道模型

离散基带模型：

$$r[n] = \sum_k h[k]s[n-k] + w[n]$$

其中：

- `s[n]`：发送符号
- `h[k]`：多径信道冲激响应
- `w[n]`：噪声
- `r[n]`：接收符号

## 3. 迫零均衡（ZF）

ZF 均衡器希望：

$$h[n] * e[n] \approx \delta[n]$$

即信道和均衡器卷积后接近单位冲激，从而消除 ISI。

ZF 的问题：如果信道频谱存在深衰落，均衡器可能显著放大噪声。

## 4. LMS 自适应均衡

LMS 使用训练序列逐步调整抽头系数，使输出和期望符号之间的均方误差减小。

更新公式：

$$e[n] = d[n] - y[n]$$

$$w[n+1] = w[n] + \mu e[n]x[n]$$

其中 `μ` 是步长。

步长影响：

- 太小：收敛慢
- 太大：可能振荡或发散

## 5. DFE 与 MLSE（拓展）

- 判决反馈均衡（DFE）：利用已判决符号抵消后续 ISI，但存在误码传播。
- MLSE/Viterbi 均衡：序列检测意义下最优，但复杂度较高。
